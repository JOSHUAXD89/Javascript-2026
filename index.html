Javascript

Javascript es un lenguaje de programación dinámico y de alto nivel para crear interfaces interactivas y dinámicas.

Es uno de los pilares fundamentales del desarrollo web.

  • 1 HTML: Define la estructura del contenido. Son los elementos que aparecen en la página web.
  • 2 CSS: Define la apariencia, colores, tamaños, estilos, etc.
  • 3 JS: Define el comportamiento, es decir lo que hace la página o sus elementos

Características de Javasvript

  1. -> Lenguaje Interpretado: No se necesita compilar. El navegador lo entiende directamente.
  2. -> Basado en onjrtos: Mnaeja los objetos de html como objetos: window, date, array, class.
  3. -> Dinámico: Podemos modificar el ontenido en tiempo real. (Sin recargar)
  4. -> Versátil y fácil de aprender: Aunque nació especialmente para la web, se usa tambié para servidores, apps móviles, videojuegos, animación y por supuesto para IA.
